Table: SFRSTCR

This table is for course enrollments. NOTE ABOUT NSU CRNs: CRNs change with terms. CRNs must always be identified as a CRN/Term Code pair. For example, 31574/202630 is a different section than 31574/202710.

Column Name Data Type Nullable Description
SFRSTCR_TERM_CODE VARCHAR2(6 CHAR) No This field identifies the registration term code.
SFRSTCR_PIDM NUMBER(8,0) No Internal Identification Number of the person registered.
SFRSTCR_CRN VARCHAR2(5 CHAR) No This field identifies the course reference number associated with the class section.
SFRSTCR_CLASS_SORT_KEY NUMBER(4,0) Yes This field identifies the class of the registrant.
SFRSTCR_REG_SEQ NUMBER(4,0) No This field identifies the sequence in which the student enrolled in the associated class section.
SFRSTCR_PTRM_CODE VARCHAR2(3 CHAR) Yes This field identifies the part-of-term code associated with this CRN.
SFRSTCR_RSTS_CODE VARCHAR2(2 CHAR) Yes This field identifies the course registration status associated with this CRN.
SFRSTCR_RSTS_DATE DATE Yes This field identifies the date associated with the course registration status of this CRN.
SFRSTCR_ERROR_FLAG VARCHAR2(1 CHAR) Yes Error associated with registration. Values: F=Fatal, D=Do not count, L=WaitListed, O=Override, W=Warning, X=Delete.
SFRSTCR_MESSAGE VARCHAR2(200 CHAR) Yes This field identifies a message associated with the registration of the CRN.
SFRSTCR_BILL_HR NUMBER(7,3) Yes This field identifies the billing hours associated with this CRN.
SFRSTCR_WAIV_HR NUMBER(7,3) Yes This field identifies the waived hours associated with this CRN.
SFRSTCR_CREDIT_HR NUMBER(7,3) Yes This field identifies the credit hours associated with this CRN.
SFRSTCR_BILL_HR_HOLD NUMBER(7,3) Yes This field identifies the billing hours to hold for reinstatements.
SFRSTCR_CREDIT_HR_HOLD NUMBER(7,3) Yes This field identifies the credit hours to hold for reinstatements.
SFRSTCR_GMOD_CODE VARCHAR2(1 CHAR) Yes This field identifies the grade mode associated with this CRN.
SFRSTCR_GRDE_CODE VARCHAR2(6 CHAR) Yes This field identifies the grade code associated with this CRN.
SFRSTCR_GRDE_CODE_MID VARCHAR2(6 CHAR) Yes This field identifies the mid-term grade code associated with this CRN.
SFRSTCR_GRDE_DATE DATE Yes This field identifies the date associated with the grade of this CRN.
SFRSTCR_DUPL_OVER VARCHAR2(1 CHAR) Yes This field identifies the duplicate course override of this CRN.
SFRSTCR_LINK_OVER VARCHAR2(1 CHAR) Yes This field identifies the link override of this CRN.
SFRSTCR_CORQ_OVER VARCHAR2(1 CHAR) Yes This field identifies the corequisite override of this CRN.
SFRSTCR_PREQ_OVER VARCHAR2(1 CHAR) Yes This field identifies the prerequisite override of this CRN.
SFRSTCR_TIME_OVER VARCHAR2(1 CHAR) Yes This field identifies the time conflict override of this CRN.
SFRSTCR_CAPC_OVER VARCHAR2(1 CHAR) Yes This field identifies the capacity override of this CRN.
SFRSTCR_LEVL_OVER VARCHAR2(1 CHAR) Yes This field identifies the level restriction override of this CRN.
SFRSTCR_COLL_OVER VARCHAR2(1 CHAR) Yes This field identifies the college restriction override of this CRN.
SFRSTCR_MAJR_OVER VARCHAR2(1 CHAR) Yes This field identifies the major restriction override of this CRN.
SFRSTCR_CLAS_OVER VARCHAR2(1 CHAR) Yes This field identifies the class restriction override of this CRN.
SFRSTCR_APPR_OVER VARCHAR2(1 CHAR) Yes This field identifies the special approval override of this CRN.
SFRSTCR_ADD_DATE DATE No This field identifies the add date of the registrant in this CRN.
SFRSTCR_ACTIVITY_DATE DATE No This field identifies the most current date record was created or updated.
SFRSTCR_LEVL_CODE VARCHAR2(2 CHAR) No This field identifies the level the registrant is enrolled for this CRN.
SFRSTCR_CAMP_CODE VARCHAR2(3 CHAR) No This field identifies the campus code of this CRN.
SFRSTCR_ATTEND_HR NUMBER(9,3) Yes This column contains the number of hours the student attended this class.
SFRSTCR_USER VARCHAR2(30 CHAR) Yes Identifies the user who entered the registration request.
SFRSTCR_WL_PRIORITY NUMBER(11,6) Yes WAITLIST PRIORITY: The Waitlist priority.
SFRSTCR_SURROGATE_ID NUMBER(19,0) No SURROGATE ID: Immutable unique key.
SFRSTCR_VERSION NUMBER(19,0) No VERSION: Optimistic lock token.
SFRSTCR_CURRENT_TIME TIMESTAMP(9) Yes CURRENT TIME: The current time of the registration.
